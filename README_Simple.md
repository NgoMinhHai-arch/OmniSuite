# OmniSuite AI - Hướng dẫn đơn giản

## 🚀 Cách chạy (KHÔNG CẦN BIẾT CODE)

### 1. Tải về
Mở Command Prompt/PowerShell, chạy:
```
git clone https://github.com/NgoMinhHai-arch/OmniSuite.git
```

### 2. Chạy
Vào folder vừa tải → **Double-click** `01_BAT_DAU_OMNISUITE.bat`

### 3. Cấu hình (lần đầu)
- Mở trình duyệt tự động hiện lên
- Click **Settings** (biểu tượng bánh răng)
- Nhập API keys của bạn (OpenAI, Gemini, v.v.)
- Save

### 4. Xài thôi!
(nếu có câu hỏi hãy hỏi Windsurf hay IDE nào đó đừng hỏi tôi nhé)
## 🗑️ Cách xóa
Xóa folder = Xóa hết (không để lại gì)

## ❓ Cần giúp?
API keys lấy ở đâu:
- OpenAI: https://platform.openai.com/api-keys
- Gemini: https://aistudio.google.com/app/apikey
- SerpAPI: https://serpapi.com/manage-api-key
